# Как сделано?

Хранение:
- CoreData + NSFetchedResultsController
- UserDefaults 
- FileManager 
- Firebase Firestore 

Многопоточность:
- GCD

Работа с сетью:
- URLSession 

Анимации: 

CAEmitters, UIViewControllerAnimatedTransitioning, CAAnimationGroup, CAKeyframeAnimation, CABasicAnimation, UIView.animate

Верстка: 
- Из кода 
- С помощью storyboard-ов и xib-ов

Тестирование: 
- Unit + UI тесты  

CI: 
- GitHub Actions
